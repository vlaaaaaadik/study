Основні CSS властивості

Поки HTML визначає вміст веб-сторінки, CSS займається її стилізацією. Тобто налаштування кольору, шрифтів, розмірів, полів, положення елементів веб-сторінки.

Властивості шрифтів

Властивість font-style - курсив

Властивість font-style дозволяє зробити текст курсивним або навпаки - скасувати курсив. Щоб зробити текст курсивом, слід додати значення italic, а щоб скасувати курсив - значення normal.

Властивість font-weight - жирність

Властивість font-weight дозволяє зробити текст жирним або навпаки - скасувати жирність (наприклад, для заголовків, які жирні за замовчуванням). Щоб зробити текст жирним, слід додати значення bold, а щоб скасувати жирність - значення normal.

Властивість font-size - розмір тексту

Властивість font-size дозволяє задати розмір тексту. Розмір задається в пікселях (позначаються px), в пунктах (позначаються pt), у відсотках (позначаються%) і за допомогою деяких інших одиниць. Між пунктами і пікселями є відповідність: 12pt = 16px

Властивість font-family - тип шрифту

Властивість font-family дозволяє задати тип шрифту (тип часто називають сімейством шрифту)

Властивість font - скорочення для шрифтів

Властивість font має наступний синтаксис: курсив жирність размер_шріфта / інтервал_между_строкамі семейство_шріфта. Обов'язковими є "размер_шріфта" і "семейство_шріфта", порядок має значення.

Властивості тексту

Властивість text-align - вирівнювання тексту

Властивість text-align дозволяє задати вирівнювання тексту. Текст можна вирівнювати по лівому краю (значення left), по правому (значення right), по центру (значення center) і одночасно і по правому, і на лівому краю (значення justify).

Властивість line-height - міжрядковий інтервал

Міжрядковий інтервал - це відстань між лініями тексту, тобто білий проміжок між ними. Властивість line-height не задає проміжок між рядками тексту, а задає висоту лінії тексту. line-height - font-size = відстань між рядками тексту

Властивість text-indent - дозволяє задати відступ першого рядка тексту

Властивість vertical-align -дозволяє задати вирівнювання по вертикалі.

Властивість color - колір тексту

Властивість color дозволяє задати колір тексту. Колір можна задавати наступними способами: англійським словом, через решітку #, через rgb і деякими іншими способами, які ми поки не будемо розбирати.

Властивості width і height - ширина і висота

Властивості width і height дозволяють задати висоту і ширину елементу відповідно. Ширина і висота зазвичай вимірюються в пікселях (позначається px) або відсотках (позначається%)
